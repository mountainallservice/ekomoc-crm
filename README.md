# Ekomoc CRM — Field-Sales CRM for Energy-Audit Teams

**Built by** [Kamil Jan](https://kamiljan.com)

A CRM built for field-sales teams running government-funded energy-audit programs.

## What it does
- **9-stage order pipeline**
- Role-based access (salesperson / auditor / admin)
- Automated **DOCX/PDF generation** for government funding contracts
- Map view, push notifications, performance leaderboard

## Stack
React · TypeScript · TanStack Router · Supabase
